=========================================================
댐 열화상 분석 MVP - 주요 코드 및 모듈 설명
=========================================================

이 문서는 협업자를 위한 주요 스크립트 및 모듈별 역할 정의서입니다.

---------------------------------------------------------
1. Blender 스크립트 (blender/scripts/)
---------------------------------------------------------

■ 01_create_dam_cells.py (실행 환경: Blender)
  - 역할: 댐의 3D 기하학적 형상 및 분석용 그리드 셀 생성
  - 주요 기능:
    * 댐 기본 형상(Body) 생성
    * 5m 단위 슬라이스 경계 및 선택용 핸들러 생성
    * 외부 사면용 직각삼각기둥 셀(Prism Cell) 생성
    * 각 셀의 고유 ID(cell_id) 및 슬라이스 ID(slice_id) 부여
    * 분석 파이프라인에서 참조할 기본 셀 정보 JSON 내보내기

■ 02_apply_thermal_result.py (실행 환경: Blender)
  - 역할: 외부 분석 결과(JSON)를 3D 모델에 시각적으로 투영
  - 주요 기능:
    * thermal_result.json 파일 로드
    * cell_id를 기준으로 Blender 내 개체 매칭
    * thermal_risk_level(0~3)에 따른 셀 색상 변경 (예: 녹색 -> 빨간색)
    * 누수 위험도가 높은 셀 강조 및 대화형 시각화 제공

---------------------------------------------------------
2. 열화상 분석 파이프라인 (src/thermal/)
---------------------------------------------------------

■ load_image.py
  - 역할: 원본 열화상 데이터 로드
  - 입/출력: PNG, JPG 또는 Radiometric CSV 로드 -> Numpy Array 변환

■ preprocess.py
  - 역할: 분석을 위한 데이터 전처리
  - 주요 기능: 노이즈 제거, 크기 조정(Resizing), 정규화, 마스크 적용, 댐 영역 크롭(Crop)

■ temperature_map.py
  - 역할: 픽셀 데이터를 실제 온도 또는 상대 온도 맵으로 변환
  - 주요 기능: Radiometric 데이터를 온도값으로 변환 (데이터 부재 시 Grayscale 비례 변환)

■ anomaly_detection.py
  - 역할: 온도 이상 구간(Anomaly) 탐지
  - 탐지 대상: 주변 대비 이상 저온/고온 영역, 급격한 온도 변화 구간, 누수 의심 구역 등

■ cell_mapping.py
  - 역할: 2D 이미지 좌표와 3D 셀 간의 공간적 매핑
  - 매핑 로직: 이미지 좌표 (u, v) -> 댐 표면 좌표 추정 -> cell_id (예: S03_X02_Z14) 연결
  - MVP 방식: 단순 비례 매핑(가로X, 세로Z)으로 시작하여 점진적 고도화

■ risk_score.py
  - 역할: 탐지된 이상 징후를 바탕으로 셀별 통합 위험도 계산
  - 계산 항목: 이상 온도 점수, 온도차(Delta T), 저온 영역 비중, 경사도 점수 등
  - 최종 출력: thermal_risk_level (0, 1, 2, 3 단계)

■ export_json.py
  - 역할: 모든 분석 결과가 통합된 최종 JSON 파일 저장
  - 저장 경로: outputs/thermal_json/thermal_result.json

---------------------------------------------------------
3. 실행 워크플로우 요약
---------------------------------------------------------

1) Blender에서 01_create_dam_cells.py 실행 (셀 생성)
2) Python 환경에서 src.thermal 파이프라인 실행 (분석 결과 생성)
3) Blender에서 02_apply_thermal_result.py 실행 (결과 시각화)
